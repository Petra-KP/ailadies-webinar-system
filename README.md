Projekt webu a podkladů pro **www.ailadies.cz** – osobní 1:1 AI mentoring od Petry Květové Pšeničné.


## Jak spustit HTML náhled

Stačí poklepat na `aplikace/index_v2.html` v Průzkumníku Windows — otevře se v prohlížeči.

Nebo v PowerShellu:

```powershell
Start-Process "01_PROJEKTY\AILadies\aplikace\index_v2.html"
```

## Redesign webu (duben 2026)

Kompletní plán úprav najdeš v [`dokumentace/AILadies_web_redesign_plan.md`](dokumentace/AILadies_web_redesign_plan.md).

Obsahuje:
1. **Analýzu** současného webu + persona check.
2. **Novou paletu** (warm terracotta) s HEX kódy pro Macaly.
3. **Fázi 1** – rychlé výhry (hero, FAQ, reference, balíčky).
4. **Fázi 2** – vizuální přestavba (bento, „pro koho JE/NENÍ", video).
5. **Fázi 3** – konverzní dotažení (lead magnet, sticky CTA, trust bar, footer).
6. **Checklist** pro implementaci v Macaly.

HTML reference (`aplikace/index_v2.html`) je hotová ukázka všech doporučení v jednom souboru — slouží jako vizuální reference a copy-paste zdroj textů při úpravě na Macaly.
